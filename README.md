## Установка и запуск
#### ТРЕБОВАНИЯ:
*go 1.24 или новее*

#### Установка
```shell
git clone https://github.com/igikawa/awesomer.git & cd awesomer
go mod tidy
go build -o awesomer cmd/tui/main.go
chmod +x ./awesomer
./awesomer
```

## Архитектура проекта

- `cmd` 
- - `tui` - Запуск 
- `internal` - Основные пакеты
- - `config` - Загрузка конфигурации из `.env`
- - `processes`- Управление процессами (чтение /proc и `syscall`)
- - `tui` - Описание и запуск консольного интерфейса
- `pkg` - Общие пакеты
- - `logger` - Логгер (пишет в awesome.log)

## Сортировки и сочетания клавиш
Утилита поддерживает сортировку процессов по разным параметрам процесса.

#### Сортировки

- `PID`: по идентификатору(в порядке возрастания) процессы сортируются
**по умолчанию** или если нажать клавишу `p`
- `Name`: по имени(в алфавитном порядке) процессы сортируются если нажать
клавишу `n`
- `CPU`: по нагрузке на процессор(в порядке убывания %) процессы сортируются
если нажать клавишу `c`
- `Mem`: по нагрузке на ОЗУ(в порядке убывания %) процессы сортируются
если нажать клавишу `m`
- `Threads`: по количеству используемых потоков(в порядке убывания) процессы
сортируются если нажать клавишу `t`

#### Выход из программы
Для выхода используется сочетание клавиш `Ctrl+c` или нажатие клавиши `q`

## Файл конфигурации

- `TICK` - определяет частоту обновления списка процессов в секундах.
Если равна 0 список будет статичным (не будет обновляться в реальном времени).
Если не задан, по у молчанию будет равен 1 секунде.
